# text-quest
### Text quest game

JavaRush Module 3 final mini-project:

https://javarush.com/quests/lectures/jru.module3.lecture04

## To build the project

In CLI use: 

```
mvn clean install
```

## To run the project

In CLI use:

```
java -jar target/text-quest-1.0.0.jar
```

or

```
mvn spring-boot:run
```
